Telegram Timer Bot

Бот для Telegram, который умеет:
- Запускать таймер по команде
- Показывать красивый прогресс-бар
- Оповещать, когда время вышло

Как установить

1. Склонируйте репозиторий: git clone https://github.com/ваш_логин/telegram-bot.git

2.Создайте файл .env (на основе .env.example): TELEGRAM_TOKEN=ваш_токен_бота

3.Установите зависимости: pip install -r requirements.txt

4.Как запустить: python main.py

5.Пример использования:
Отправьте боту сообщение с временем:
10s  # таймер на 10 секунд
2m   # таймер на 2 минуты
